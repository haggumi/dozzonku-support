# 개인정보처리방침

본 앱은 **회원가입 기능이 없으며**, 다음과 같은 개인정보를 **수집·저장·이용하지 않습니다**.

* 이름, 이메일, 전화번호 등 개인 식별 정보

* 기타 개인을 식별할 수 있는 정보

## 2. 자동 수집 정보

본 앱은 서비스 제공 과정에서 **기기 식별이 가능한 정보(IDFA, Android ID 등)를 수집하거나 사용자와 연결하지 않습니다.**

※ 단, 앱의 안정성 향상 및 오류 분석을 위해 Google Play / Apple App Store에서 제공하는 **비식별 통계 정보**가 자동으로 수집될 수 있으나, 이는 개인을 식별할 수 없습니다.
